### Usage

```jsx
import { Blockquote } from 'deichman-ui';

<Blockquote cite="Arthur C. Clarke">Any sufficiently advanced technology is indistinguishable from magic.</Blockquote>;
```

#### Without React:

```html
<blockquote className="blockquote">
  <div className="blockquote__text">
    «Any sufficiently advanced technology is indistinguishable from magic.»
  </div>
  <footer className="blockquote__footer">Arthur C. Clarke</footer>
</blockquote>
```
