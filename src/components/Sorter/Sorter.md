### Usage

```jsx
import { Sorter } from 'deichman-ui';

<Sorter
  active={true}
  direction="asc"
  onClick={() => {}}
>;
```

### Props/modifiers:

| prop name | type   | Comments             |
| --------- | ------ | -------------------- |
| active    | bool   | Highlights the arrow |
| direction | string | "asc" or "des"       |
| onClick   | func   |                      |
