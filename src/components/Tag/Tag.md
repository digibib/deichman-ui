### Usage

```jsx
import { Tag } from 'deichman-ui';

<Tag text="Hi there" />;
```

#### Without React:

```html
<span class="tag">Hi there</span>
```

### Props/modifiers:

| prop name | type   | without react |
| --------- | ------ | ------------- |
| text      | string | N/A           |
| active    | bool   | N/A           |
| showClear | bool   | N/A           |
| disabled  | bool   | N/A           |
| link      | string | N/A           |
| onClick   | func   | N/A           |
| onClear   | func   | N/A           |
