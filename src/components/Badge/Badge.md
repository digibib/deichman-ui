### Usage

```jsx
import { Badge } from 'deichman-ui';

<Badge text="Hi there" />;
```

#### Without React:

```html
<span class="badge">Hi there</span>
```

### Props/modifiers:

| prop name | type   | without react |
| --------- | ------ | ------------- |
| text      | string | N/A           |
| theme     | string | .col-ok-{1-6} |
